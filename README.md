> A sleek, feature-rich streaming platform front-end with watchlist, history, ratings, and continue-watching functionality — all powered by vanilla JavaScript and the OMDB API.

![StreamVault Demo](https://via.placeholder.com/1200x600?text=StreamVault+Demo+Screenshot)

## ✨ Features

### Core Experience
- 🎥 **Hero Carousel** — Auto-playing spotlight with manual swipe/dot navigation
- 🔍 **Real-time Search** — Instant results with movie/series/episode filters
- 📱 **Fully Responsive** — Optimized for mobile, tablet, and desktop with bottom navigation bar

### User Features
- 📌 **Watchlist** — Save titles to watch later with visual badges
- 🕐 **Watch History** — Automatically track everything you've watched
- ▶️ **Continue Watching** — Resume where you left off with progress indicators
- ⭐ **User Ratings** — Rate titles 1-10 stars, view your rated content
- ⚙️ **Preferences** — Toggle autoplay, history tracking, and IMDb ratings display

### Media Player
- 🎬 **Embed Player** — Integrated video player for movies and TV series
- 📺 **Season Navigation** — Episode picker for multi-season shows
- 🔗 **Share & Embed** — Copy IMDb links or embed URLs to share content

### Data Persistence
- 💾 **Local Storage** — All watchlist, history, ratings, and preferences saved locally
- 🚀 **OMDB Caching** — API results cached for better performance
- 🗑️ **Data Management** — Clear history, continue watching, or wipe all data

## 🛠️ Tech Stack

| Technology | Purpose |
